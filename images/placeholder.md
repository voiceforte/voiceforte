# Image Placeholders

Replace these with your actual images:

1. logo.png (500x500px) - Website logo
2. hero-bg.jpg (1920x1080px) - Hero background
3. founder.jpg (800x1000px) - Founder photo
4. team/ - Team member photos
5. portfolio/ - Project portfolio images
6. clients/ - Client logos

## Recommended Image Sizes:
- Logo: 500x500px (PNG with transparent background)
- Hero images: 1920x1080px (JPG/WebP)
- Team photos: 800x1000px (JPG)
- Portfolio: 1200x800px (JPG/WebP)
- Client logos: 300x200px (PNG)

## Optimization Tips:
- Use WebP format for better compression
- Compress images before uploading
- Use descriptive filenames
- Add alt text for accessibility
